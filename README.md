## 📒 프로젝트 소개
Linkhub는 다양한 사람들이 자신의 북마크를 주제에 맞게 등록하고 서로 공유할 수 있도록 하는 일종의 북마크 저장소 입니다.<br/><br/>
기능은 다음과 같습니다. <br/>
1. 검색을 통해 북마크를 찾을 수 있습니다.
2. 북마크를 작성 및 조회 할 수 있습니다.
3. 원하는 사이트로 쉽고 간편하게 이동할 수 있습니다.
<br/><br/>
